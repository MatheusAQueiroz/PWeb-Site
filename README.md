#Legenda de Versionamento

##Significado dos números do versionamento (e.g. v.0.5.1)

Primeiro: Avaliação destino (Só pode ser 1 ou 2);
Segundo: Quantidade de telas finalizadas;
Terceiro: Índice do commit.

Sendo assim no exemplo dado acima o site se encontrava incompleto para a P1, com 5 telas prontas e sendo o primeiro commit com 5 telas prontas.
